CRM Customer

A small REST API built in Go to manage customers (Customer) with basic CRUD (Create, Read, Update, Delete) operations. The application uses an in-memory repository and allows you to manage both individual and multiple clients at the same time.

The port for launch api is 3333

  • Get all customers
  • Get customer by id
  • Create a new customer
  • Create several customers
  • Update a customer
  • Delete a customer
